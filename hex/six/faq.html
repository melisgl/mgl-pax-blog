FAQ

What is Six?

Six is a Hex playing program for KDE.

What is Hex?

Hex is great game. It belongs to the family of connectivity games. Its simple rules and deep strategic complexity make it a true classic. See this category for more information.

What license is Six under?

Six is GNU GPL'd.

How strong is the AI?

I think it is one of the strongest AIs around on board of reasonable size (greater than 9). On small boards where the game tree size is managable, it is still strong, but there is no guarantee of optimal play.

How does the AI work?

The basic idea is described by Vadim Anshelevich in his publications.

Does it run under Windows?

No.
